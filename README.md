# a-frame-webpack-boilerplate
Minimalist boilerplate for A-Frame 1.3.0 and Webpack 5

Create a folder for your project and : 

### clone
```sh
git clone https://github.com/Chabloz/a-frame-webpack-boilerplate.git .
```sh
### Install
```sh
npm install
```
### Dev
```sh
npm run dev
```
### Build
```sh
npm run build
```
